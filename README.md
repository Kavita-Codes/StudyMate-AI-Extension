# 🚀 StudyMate AI - AI Reading & Learning Assistant

> A powerful AI-powered Chrome Extension that helps students, developers, researchers, and professionals summarize, explain, translate, and interact with any webpage using AI.

# ✨ Features

## 📖 AI Smart Summary

Generate multiple types of summaries:

- Short Summary
- Bullet Summary
- Key Takeaways
- Exam Notes
- LinkedIn Summary
- Quick Revision

---

## 🧠 Explain Selected Text

Select any text on any webpage and get:

- Beginner Explanation
- Detailed Explanation
- Hinglish Explanation
- Real-life Examples
- Technical Explanation

---

## 🌍 AI Translator

Translate selected text into multiple languages.

Supported Languages:

- English
- Hindi
- French
- German
- Japanese
- Spanish

---

## 💬 Ask AI

Ask custom questions about selected text.

Examples:

- Explain this
- Why?
- Give Examples
- Generate Interview Questions
- Convert into Notes
- Simplify this
- Explain in Hinglish

---

## 📚 AI Notes Generator

Convert long articles into:

- Study Notes
- Revision Notes
- Key Points
- Action Items

---

## 📝 Quiz Generator

Generate AI-powered quizzes from any article.

- MCQs
- Short Questions
- Interview Questions

---

## 📌 Save Highlights

Save important content for later.

Features:

- Local Storage
- Search
- Categories
- Favorites

---

## ✨ Copy Cleaner

Clean copied text by removing:

- Extra Spaces
- HTML Tags
- Formatting Issues
- Line Breaks

---

## 🌙 Modern UI

- Responsive Popup
- Dark Mode
- Beautiful Animations
- Glassmorphism Design
